# 需求分析

主要功能：

1. 在网络下上传和下载文件
2. 分配用户权限， . 普通用户只可以从库中读取和下载文件 .
   管理员用户可以上传和修改库中的文件
3. 主要通过http访问网站的形式交互
4. 支持多线程下载
5. 使用连接池进行连接的分配和回收
6. 日志记录
7. 在文件搜索时，搜索通过字典树实现

日志功能：

1. 日志格式 ：日志级别 + time/day/month/year + message
2. 日志输出到具体文件
3. 单文件日志大小 超过了创建新的日志文件 设置具体时间 更换日志

拓展:

1. 用户登录 注册
2. 用户之间传输文件
3. ......
